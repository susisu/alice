# Alice
Alice sings when you type `al`.

![screenshot](https://github.com/susisu/alice/wiki/images/al_ja.gif)

## Requirements
* Node.js (use latest version)

## Install
``` shell
npm install -g https://github.com/susisu/alice.git
```

## Why don't you sing in English?
``` shell
al -e
```
